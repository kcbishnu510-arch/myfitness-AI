# MyFitnessAI

A personalized fitness and nutrition assistant powered by AI.

## Features
- Personalized workout plans based on your fitness level and goals
- Custom nutrition recommendations with macronutrient calculations
- AI-powered fitness assistant for ongoing guidance
- User-friendly interface with responsive design

## Technologies Used
- Next.js 14 with App Router
- React 18
- TypeScript
- Tailwind CSS
- Gemini AI API

## Getting Started
1. Clone the repository
2. Install dependencies with `npm install`
3. Copy `.env.example` to `.env.local` and add your Gemini API key
4. Run the development server with `npm run dev`
## Usage

1. Enter your personal details and fitness goals
2. Calculate your nutrition requirements
3. Generate personalized workout plans
4. Use the AI chat assistant for fitness guidance
